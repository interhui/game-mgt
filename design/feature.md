# 游戏管理程序 - 功能与特性

## 1. 概述

游戏管理程序是一款基于 Electron 框架开发的模拟器游戏管理工具，用于集中管理多个平台的游戏收藏。

## 2. 主要功能

### 2.1 游戏库管理

- **多平台支持**：支持 PlayStation 2、PlayStation、PlayStation Portable、Xbox 360、Nintendo Switch、PC Games 等多个游戏平台
- **游戏扫描**：自动扫描游戏目录，发现并加载游戏信息
- **游戏信息展示**：显示游戏海报、名称、平台、发行日期、评分、收藏状态等
- **游戏搜索**：支持按名称、描述、标签搜索游戏
- **游戏排序**：支持按名称、评分、最后游戏时间等字段排序

### 2.2 平台筛选与联动

- **侧边栏筛选**：在左侧平台列表中点击可筛选特定平台
- **下拉框筛选**：顶部下拉框同样支持平台筛选
- **双向联动**：侧边栏选择平台时，下拉框同步更新；反之亦然

### 2.3 游戏详情查看

- **详情窗口**：点击游戏卡片打开独立详情窗口
- **信息展示**：显示完整游戏信息，包括海报、描述、评分、标签等
- **编辑功能**：可编辑游戏名称、发行日期、发行商、标签、描述等
- **评分系统**：支持 1-5 星评分
- **收藏功能**：支持标记/取消标记游戏为收藏

### 2.4 游戏盒子

- **盒子创建**：可创建多个游戏盒子（如"我的收藏"、"待玩游戏"等）
- **添加游戏**：可将游戏添加到指定盒子
- **批量添加**：支持勾选多个游戏后批量添加到盒子
- **盒子浏览**：在新窗口中浏览特定盒子中的游戏
- **盒子统计**：显示盒子中游戏总数、各状态游戏数量

### 2.5 游戏盒子详情

- **状态跟踪**：记录游戏状态（未玩、游戏中、已玩、已完成）
- **游戏时间**：记录累计游戏时长
- **最后游戏**：记录最后游玩日期
- **状态修改**：支持点击状态标签快速修改游戏状态
- **批量移除**：支持在列表视图中勾选多个游戏后批量移除

### 2.6 游戏启动

- **模拟器集成**：可配置各平台的模拟器路径和启动参数
- **快速启动**：通过详情页按钮启动游戏

### 2.7 设置功能

- **主题切换**：支持深色/浅色主题
- **目录配置**：可配置游戏目录和游戏盒子目录
- **模拟器配置**：可配置各平台对应的模拟器
- **布局调整**：可调整侧边栏宽度、海报大小、视图模式

## 3. 视图模式

### 3.1 网格视图

- 以卡片形式展示游戏，带海报图片
- 显示游戏名称、平台图标、收藏标记、评分

### 3.2 列表视图

- 以表格形式展示游戏
- 显示游戏ID、名称、描述、发行日期、平台、发行商、评分等
- 支持复选框多选

## 4. 快捷键

| 快捷键 | 功能 |
|--------|------|
| Ctrl+F | 打开搜索 |
| Ctrl+K | 聚焦搜索框 |
| Enter | 启动游戏 |
| Ctrl+D | 游戏详情 |
| Ctrl+E | 编辑游戏 |
| Delete | 删除游戏 |
| F | 切换收藏 |
| R | 刷新游戏库 |
| Ctrl+, | 打开设置 |

## 5. 技术特性

- **Electron 框架**：跨平台桌面应用框架
- **多窗口架构**：主窗口、详情窗口、盒子窗口独立运行
- **IPC 通信**：主进程与渲染进程安全通信
- **contextBridge**：安全的 API 暴露机制
- **JSON 数据存储**：游戏信息存储在游戏目录的 game.json 文件中
- **日志系统**：使用 electron-log 记录运行日志
