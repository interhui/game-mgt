# 游戏管理程序 - 用户操作手册

## 1. 概述

游戏管理程序是一款基于 Electron 的模拟器游戏管理工具，用于集中管理多个平台的游戏收藏，支持游戏分类、搜索、评分、收藏、游戏盒子等功能。

## 3. 基础操作

### 3.1 查看游戏

1. 在**侧边栏**点击平台名称，可筛选该平台下的所有游戏
2. 或使用**顶部平台下拉框**进行筛选
3. 侧边栏和下拉框会**联动**——选择一个，另一个会自动同步

### 3.2 搜索游戏

1. 在**搜索框**中输入游戏名称、描述或标签关键词
2. 点击**搜索按钮**或按 **Enter** 键进行搜索
3. 搜索结果会即时显示在游戏区域

### 3.3 查看游戏详情

1. **点击游戏卡片**，会打开一个新的详情窗口
2. 详情窗口显示游戏的完整信息：海报、名称、ID、平台、发行日期、发行商、评分、标签、描述等
3. 在详情窗口中可以进行评分、编辑、删除等操作

### 3.4 评分游戏

1. 打开游戏详情页
2. 点击**星星图标**进行评分（1-5星）
3. 评分会自动保存

### 3.5 收藏游戏

1. 打开游戏详情页
2. 点击**收藏按钮**（❤️/🤍）
3. 已收藏的游戏会在游戏卡片上显示❤️标记

## 4. 游戏管理

### 4.1 编辑游戏信息

1. 打开游戏详情页
2. 点击**编辑按钮**
3. 进入编辑模式，字段变为可编辑状态
4. 可编辑：名称、发行日期、发行商、标签、描述
5. 点击**确定**保存修改，或点击**取消**放弃修改

### 4.2 删除游戏

1. 打开游戏详情页
2. 点击**删除按钮**
3. 在确认对话框中点击**确定**
4. 游戏会从游戏库中删除（仅删除 game.json，不删除游戏文件）

### 4.3 切换视图模式

1. 点击工具栏右侧的**视图切换按钮**
2. 可在**网格视图**和**列表视图**之间切换
3. 列表视图支持多选，便于批量操作

## 5. 游戏盒子

### 5.1 创建游戏盒子

1. 在**侧边栏底部**点击**添加盒子按钮**（+图标）
2. 或通过菜单：**文件 → 新建盒子**
3. 输入盒子名称
4. 点击**确定**创建

### 5.2 添加游戏到盒子

**单个添加：**
1. 打开游戏详情页
2. 点击**添加按钮**
3. 在弹出的盒子选择框中选择目标盒子
4. 点击**确定**

**批量添加：**
1. 在**列表视图**中勾选多个游戏（复选框）
2. 点击**批量添加按钮**（显示已选数量）
3. 在弹出的盒子选择框中选择目标盒子
4. 点击**确定**

### 5.3 浏览游戏盒子

1. 在**侧边栏的游戏盒子列表**中点击盒子名称
2. 会**打开一个新窗口**显示该盒子的游戏

### 5.4 修改盒子中的游戏状态

**在盒子视图中：**
1. 在**网格视图**中，点击游戏卡片右上角的状态标签
2. 或在**列表视图**中点击状态列
3. 在弹出的状态选择框中选择新状态
4. 点击**确定**

**在详情页中：**
1. 打开盒子中的游戏详情页
2. 点击**修改状态按钮**
3. 在弹出的状态选择框中选择新状态
4. 点击**确定**

### 5.5 移除游戏（从盒子）

**单个移除：**
1. 打开游戏详情页（来自盒子）
2. 点击**移除按钮**
3. 在确认对话框中点击**确定**

**在盒子列表中移除：**
1. 在**网格视图**中，点击游戏卡片上的**X按钮**
2. 或在**列表视图**中点击操作列的**X按钮**

### 5.6 批量移除游戏

1. 在**列表视图**中勾选要移除的游戏
2. 点击**批量移除按钮**
3. 在确认对话框中点击**确定**

### 5.7 删除游戏盒子

1. 在**侧边栏**右键点击盒子名称
2. 选择**删除盒子**
3. 在确认对话框中点击**确定**

## 6. 设置

### 6.1 打开设置

- 点击工具栏右侧的**设置按钮**
- 或使用快捷键 **Ctrl+,**

### 6.2 主题设置

1. 在设置面板中选择**主题**
2. 可选：`dark`（深色）或 `light`（浅色）
3. 主题切换即时生效

### 6.3 目录设置

1. **游戏目录**：存储游戏文件夹的目录
   - 点击**选择**按钮可浏览选择新目录
2. **游戏盒子目录**：存储盒子配置文件的目录
   - 点击**选择**按钮可浏览选择新目录

### 6.4 模拟器配置

1. 设置 → 模拟器
2. 为每个平台配置：
   - **名称**：模拟器显示名称
   - **路径**：模拟器可执行文件路径
   - **参数**：启动参数，`{gamePath}` 会被替换为游戏路径

### 6.5 布局设置

- **侧边栏宽度**：调整侧边栏宽度（像素）
- **海报大小**：选择 `small`、`medium`、`large`
- **视图模式**：默认 `grid` 或 `list`

### 6.6 保存设置

1. 完成设置修改后
2. 点击**保存**按钮
3. 设置会立即生效

## 8. 菜单栏

### 8.1 文件菜单

- **刷新游戏库**：重新扫描游戏目录
- **设置**：打开设置面板
- **退出**：退出应用程序

### 8.2 视图菜单

- **重新加载**：重新加载页面
- **强制重新加载**：强制重新加载（清除缓存）
- **开发者工具**：打开开发者工具
- **缩放**：放大/缩小/重置缩放
- **全屏**：切换全屏模式

### 8.3 帮助菜单

- **关于**：显示应用程序信息

## 9. 游戏状态说明

| 状态 | 标识 | 说明 |
|------|------|------|
| unplayed | 未玩 | 尚未开始游玩的游戏 |
| playing | 游戏中 | 当前正在游玩的游戏 |
| played | 已玩 | 已经玩过但未完成的游戏 |
| completed | 已完成 | 已完全通关的游戏 |

## 10. 数据存储

- **游戏信息**：存储在每个游戏文件夹的 `game.json` 文件中
- **游戏盒子**：存储在 `boxes` 目录下的 `{盒子名}.json` 文件中
- **应用设置**：存储在 `config/settings.json` 文件中

## 11. 常见问题

### Q: 游戏没有显示在列表中
A: 请确认：
1. 游戏目录配置正确（设置 → 目录）
2. 每个游戏文件夹中有 `game.json` 文件
3. `game.json` 中的 `platform` 字段与文件夹名称匹配

### Q: 无法启动游戏
A: 请确认：
1. 已正确配置模拟器路径（设置 → 模拟器）
2. 模拟器文件存在且可执行

### Q: 盒子中的游戏状态修改没有保存
A: 确保点击确定按钮后等待操作完成，数据会自动保存到盒子 JSON 文件中。

### Q: 批量添加游戏到空盒子失败
A: 确保盒子目录存在且有写入权限。如果问题持续，请检查盒子 JSON 文件是否损坏。
