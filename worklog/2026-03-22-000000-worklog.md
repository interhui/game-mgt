# Worklog: 修复游戏详情编辑模式按钮问题

## 日期
2026-03-22

## 问题描述

### 原始问题
在游戏详情的编辑模式下，可以选择其他游戏，导致游戏详情中同时出现了：编辑、删除、添加、确定和取消五个按钮。

### 问题分析
通过代码分析，发现问题原因如下：

1. **按钮切换逻辑错误**：在 `detail.js` 的 `enterEditMode()` 函数中，代码正确地隐藏了 `normalActions` 和 `boxActions`，并显示 `editActions`。但是当用户在编辑模式下点击主窗口的游戏卡片时，会触发 `loadGameDetail()` 函数加载新游戏数据，该函数会无条件地根据 `fromBox` 标志显示 `normalActions` 或 `boxActions`，覆盖了编辑模式的按钮设置。

2. **缺少编辑模式锁定机制**：详情窗口在编辑模式下时，主窗口和盒子窗口仍然可以发送新的游戏数据到详情窗口，导致编辑状态被中断。

3. **缺少关闭确认**：在编辑模式下点击关闭按钮时，没有提示用户未保存的更改将被丢弃。

## 解决方案

### 1. 修改 `loadGameDetail()` 函数
- 添加 `isEditMode` 检查
- 如果在编辑模式下尝试加载新游戏，弹出确认对话框询问用户是否放弃更改
- 如果用户确认，则调用 `discardChangesAndSwitch()` 放弃更改并切换游戏

### 2. 修改关闭按钮处理逻辑
- 在编辑模式下点击关闭按钮时，弹出确认对话框
- 用户确认后关闭详情窗口并通知主窗口解锁
- 用户取消则继续处于编辑模式

### 3. 添加编辑模式锁定机制
- 在 `enterEditMode()` 时通知主窗口锁定游戏卡片点击
- 在 `exitEditMode()` 或 `discardChangesAndSwitch()` 时通知主窗口解锁
- 主窗口和盒子窗口监听锁定状态，在锁定时忽略游戏卡片点击事件

### 4. 添加新的 IPC 通道
- 添加 `set-detail-edit-mode` IPC 处理器
- 添加 `detail-edit-mode-changed` 事件通道

## 用户交互流程

### 编辑模式下切换游戏的流程
1. 用户在游戏详情窗口中点击"编辑"进入编辑模式
2. 主窗口收到通知，游戏卡片点击被锁定
3. 如果用户在主窗口点击游戏卡片，显示提示"游戏详情窗口处于编辑模式，请先保存或取消编辑"
4. 如果用户在详情窗口点击"确定"保存更改，编辑模式退出，游戏数据更新
5. 如果用户在详情窗口点击"取消"放弃更改，编辑模式退出，游戏数据恢复
6. 如果用户在编辑模式下尝试切换游戏，弹出确认对话框

### 编辑模式下关闭窗口的流程
1. 用户在编辑模式下点击关闭按钮（×）
2. 弹出确认对话框："当前编辑未保存，是否强制退出？"
3. 用户点击"确定"，窗口关闭，主窗口解锁
4. 用户点击"取消"，窗口保持打开状态，继续处于编辑模式

## 核心代码变更

### detail.js - loadGameDetail() 修改
```javascript
function loadGameDetail(game) {
    // 如果当前处于编辑模式，不允许切换游戏
    if (isEditMode) {
        pendingGameData = game;
        const switchConfirmed = confirm('当前有未保存的编辑，是否放弃更改并切换到选中的游戏？\n\n点击"确定"放弃更改，点击"取消"继续编辑当前游戏。');
        if (switchConfirmed) {
            discardChangesAndSwitch(game);
        } else {
            pendingGameData = null;
        }
        return;
    }
    // ... 原有逻辑
}
```

### detail.js - 关闭按钮修改
```javascript
elements.closeBtn.addEventListener('click', () => {
    if (isEditMode) {
        const confirmed = confirm('当前编辑未保存，是否强制退出？');
        if (confirmed) {
            window.electronAPI.setDetailEditMode(false);
            window.close();
        }
    } else {
        window.close();
    }
});
```
