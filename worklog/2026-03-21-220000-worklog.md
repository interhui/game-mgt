# Worklog: 应用启动后窗口最大化

## 问题描述
用户需求：应用启动之后，窗口大小使用最大化

## 分析过程

### 1. 代码分析
通过阅读 `main.js` 文件，找到主窗口创建逻辑：

```javascript
function createMainWindow() {
    mainWindow = new BrowserWindow({
        width: 1280,
        height: 800,
        minWidth: 800,
        minHeight: 600,
        title: '游戏管理程序',
        // ...
        show: false
    });

    // 显示窗口后ready-to-show
    mainWindow.once('ready-to-show', () => {
        mainWindow.show();
        log.info('Main window displayed');
    });
}
```

### 2. 解决方案
Electron 的 BrowserWindow 提供了 `maximize()` 方法，可以在窗口显示后调用使其最大化。

在 `ready-to-show` 事件中，调用 `mainWindow.maximize()` 即可实现启动后最大化。

## 修改内容

### 文件: main.js
**位置**: `createMainWindow()` 函数中的 `ready-to-show` 事件处理

**修改前**:
```javascript
mainWindow.once('ready-to-show', () => {
    mainWindow.show();
    log.info('Main window displayed');
});
```

**修改后**:
```javascript
mainWindow.once('ready-to-show', () => {
    mainWindow.show();
    mainWindow.maximize();
    log.info('Main window displayed and maximized');
});
```
