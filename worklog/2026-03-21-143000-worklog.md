## 任务概述
实现从 IGDB（Internet Game Database）导入游戏的功能，用户可以通过 IGDB API 搜索游戏信息并添加到游戏库中。

## 需求分析

### 用户流程
1. 用户点击菜单"游戏" -> "从 IGDB 导入..."
2. 弹出 IGDB 导入对话框
3. 用户输入游戏名称，点击"搜索"
4. 系统调用 IGDB API 获取游戏信息并展示
5. 用户选择平台，点击"确认添加"完成游戏添加

### IGDB API 调用流程
1. **获取 Access Token**
   - API: POST https://id.twitch.tv/oauth2/token
   - Query 参数: client_id, client_secret, grant_type=client_credentials
   - 响应: { access_token, expires_in, token_type }
   - Token 缓存在内存中

2. **搜索游戏**
   - API: POST https://api.igdb.com/v4/games
   - Header: Client-ID, Authorization: Bearer {access_token}
   - Body: `search "game name"; fields name,summary,platforms.name,videos,language_supports.language,cover.url,screenshots,release_dates,involved_companies.company.name;`

3. **数据转换**
   - name -> 游戏名称
   - summary -> 游戏描述
   - platforms.name -> 平台列表
   - first_release_date / release_dates -> 发行日期 (yyyy-MM-dd)
   - cover.url -> 封面 URL
   - involved_companies[0].company.name -> 发行商

## 配置说明

### 获取 IGDB API 凭证
1. 访问 https://dev.twitch.tv/console/apps
2. 注册/登录 Twitch 开发者账号
3. 创建新应用，获取 Client ID 和 Client Secret
4. 在应用设置中填入凭证

### 使用流程
1. 打开应用设置
2. 在"IGDB API"区域填入 Client ID 和 Client Secret
3. 保存设置
4. 点击菜单"游戏" -> "从 IGDB 导入..."
5. 输入游戏名称进行搜索
6. 选择游戏和平台，点击确认添加

## 注意事项
1. IGDB API 请求需要有效的 Twitch OAuth Token
2. Token 会在过期前自动刷新
3. 搜索报文体严格按照要求格式，不可修改
4. 封面图片 URL 需要添加 `https:` 前缀才能正常显示
