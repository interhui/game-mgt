2025-3-14
制作一份软件详细设计文档，将设计文档已Markdown格式写入：Design.md文件中，游戏需求如下：
开发语言：node.js
开发框架：Electron
说明：模拟器游戏管理程序，主要功能参考playnite

程序目录结构
|--image，程序图片
|--games，游戏存储目录
|--styles，CSS样式

游戏存储目录结构：
1、games下的文件夹，games下的文件夹是模拟器文件夹名称，如ps1、ps2、psp、xbox360等
2、在模拟器文件夹下，包括具体的游戏文件夹，如Superman01
3、在游戏文件夹下，有游戏的封面文件：poster.jpg，游戏基本信息：game.json
举例如下：
games
 |--ps2
      |--superman
           |--poster.jpg
           |--game.json

game.json内容如下：
{
    "name": "Superman",
    "description": "A Superman RPG Game",
    "publishDate": "2021-10-12",
    "platform": "ps2"
}

功能特性：
1、游戏展示主界面采用左右结构，左边为模拟器列表，右边为游戏海报墙，点击左侧模拟器名称，右侧海报墙根据模拟器类型进行筛选
2、海报墙展示游戏时，读取所有游戏文件夹下的poster.jpg文件作为海报进行展示，在1920*1080分辨率下，每一行可以展示6个游戏海报，海报下方包括游戏名称和模拟器名称，游戏名称和模拟器名称从game.json文件中读取，game.json中name为游戏名称, platform为平台名称，展示模拟器名称时候，需要携带模拟器的小图标；
3、点击游戏海报之后进入游戏详情，左侧展示游戏海报，右侧展示游戏信息(从game.json中读取)，包括游戏名称(name)、游戏描述(description)、发行时间(publishDate)
4、支持游戏搜索功能，支持输入游戏关键字进行搜索

增加新特性：
1、允许用户创建自己的游戏盒子（Game Box），可以在游戏列表中选择多个游戏，然后将选择的游戏加入到游戏盒子中。游戏盒子仅保存游戏ID，并以在box文件夹下建立游戏盒子命名的JSON文件，例如：
创建游戏盒子：MyBox，则在box文件夹中创建MyBox.json，并且在游戏管理页面中选择“ps2-gta-san-andreas”，“ps2-final-fantasy-x”，“xbox360-mass-effect-2”，“xbox360-red-dead-redemption”四个游戏加入游戏盒子中

创建内容为：
{
    "ps2": [

            "status": "played",
            "firstPlayed": "2024-01-20",
            "lastPlayed": "2026-10-24",
            "totalPlayTime": 126,
            "playCount": 4
        },
        {
            "id": "ps2-final-fantasy-x",
            "status": "playing",
            "firstPlayed": "2024-01-20",
            "lastPlayed": "2026-10-24",
            "totalPlayTime": 130,
            "playCount": 5
        }
    ],
    "xbox360": [
        {
            "id": "xbox360-mass-effect-2",
            "status": "unplayed",
            "firstPlayed": "",
            "lastPlayed": "",
            "totalPlayTime": 0,
            "playCount": 0
        },
        {
            "id": "xbox360-red-dead-redemption",
            "status": "completed",
            "firstPlayed": "2024-03-25",
            "lastPlayed": "2026-08-13",
            "totalPlayTime": 240,
            "playCount": 11
        }
    ]
}
其中字段内容含义
id：游戏ID
status：游戏状态：played（已玩），playing（游戏中），unplayed（未玩），completed（完成）
firstPlayed：首次游戏
lastPlayed：最后游戏
totalPlayTime：游戏时间（分钟）
playCount：游戏次数
